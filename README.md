# MagicApp_API
Proyecto Final de Codenotch

MySQL  
```
1. Crear el proyecto con npm init
2. Instalar express: npm i express
3. Instalar cors: npm i cors
4. Crear la carpeta src
5. Crear la carpeta src/controller
6. Crear la carpeta src/error
7. Crear la carpeta src/routers
8. Crear el fichero src/controller/user.controller.js
9. Crear el fichero src/error/errorHandling.js
10. Crear el fichero src/routers/user.routers.js
11. Crear los ficheros src/app.js y src/apiRest.js
```

Mongo  
```
Crear el proyecto con npm init
2. Instalar express: npm i express
3. Instalar cors: npm i cors
4. Instalar mongoose: npm i mongoose
5. Crear las carpetas: src, src/controller, src/error, src/routers
6. Crear el fichero src/controller/user.controller.js
7. Crear el fichero src/error/errorHandling.js
8. Crear el fichero src/routers/user.routers.js
9. Crear los ficheros src/app.js y src/apiRest.js
10. Crear el fichero src/model/user.js
11. Crear el fichero src/database.js
```
